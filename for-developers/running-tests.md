# Running tests

***

## Unit tests

<pre class="language-bash"><code class="lang-bash"><strong>secator test unit
</strong></code></pre>

{% hint style="info" %}
Run `secator test unit --help` for more options.

Run `secator test integration --help` for more options.
{% endhint %}
